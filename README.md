# numerimap

This library handles maps with numeric values, e.g. {:a 1 :b 2} that can be manipulated mathematically.

## Usage

```
(m-add {:a 1 :b 2} {:a 3 :b 4}) => {:a 4 :b 6}
```

The other functions should be obvious.

* m-add
* m-sub
* m-mul
* m-div
* m-map
